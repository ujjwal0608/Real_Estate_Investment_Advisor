# 🏠 Real Estate Investment Advisor

An ML-powered web application that analyzes real estate properties and predicts future prices to help users make smarter investment decisions.

---

## 🚀 Features

- 📊 **5-Year Price Prediction**
- 📈 **Investment Recommendation (Good/Average)**
- 📉 **Annual Growth Rate (CAGR)**
- 📊 **Interactive Charts & Visualizations**
- 🏙️ **Market Insights & Trends**
- 🔍 **Data Explorer with Filters**
- 📄 **Downloadable Investment Report**

---

## 🧠 Machine Learning Models

- **Regression Model:** Gradient Boosting Regressor  
  → Predicts future property prices  

- **Classification Model:** XGBoost Classifier  
  → Predicts whether investment is good or not  

---

## 🛠️ Tech Stack

- **Frontend/UI:** Streamlit  
- **Backend:** Python  
- **ML Libraries:** Scikit-learn, XGBoost  
- **Visualization:** Plotly, Matplotlib, Seaborn  
- **Data Handling:** Pandas, NumPy
